# AIJavaPatterns

Java examples and patterns for AI applications.

## Modules

- `a2a-protocol`: A2A Java SDK + Ollama PoC with two local agents that route and delegate requests.

## Build

Run all tests from the repository root:

```bash
mvn test
```

Run only the current module:

```bash
mvn -pl a2a-protocol test
```
